# RAI Streaming VLC

Script per vedere gli stream delle dirette RAI usando VLC (non serve Flash).

## Utilizzo

`python diretta.py`

## Dipendenze

* `python-requests`
* `vlc` nel PATH

(Scritto per Linux, ma con una minima modifica al percorso di vlc dovrebbe funzionare anche su Windows.)
